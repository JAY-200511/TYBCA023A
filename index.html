Introduction to Ciphers

Caesar Cipher

A simple substitution cipher that uniformly shifts each letter of the alphabet by a fixed number of places.

Monoalphabetic Cipher

A substitution cipher where each letter of the alphabet is consistently mapped to a different, unique letter using a shuffled alphabet key.

Transposition Cipher

A cipher that scrambles the order of the letters in the message, rather than changing the letters themselves, using a keyword to define the new order.
